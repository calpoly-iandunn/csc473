Textbook

The recommended, but optional, textbook for this course is Physically Based Rendering.

Project

Overview

Throughout this quarter you will implement the basic functions of a distributed ray tracer. This software will be enhanced throughout the quarter, thus this initial assignment will serve as the base for following assignments. Since you will be adding and reusing your code, it is advised that you write your code in a clean, structured object-oriented fashion. All code must be written in C++.

Milestones

  1. Program 1 - File Parsing and Ray Cast due 4/14.
  2. Program 2 - Shading, Shadows, and Camera due 4/28.
  3. Program 3 - Reflection, Refraction, and Triangles due 5/12.
  4. Program 4 - Transforms and Anti-aliasing due 5/19.
  5. Program 5 - Spatial Data Structures due 5/26.
  6. Program 6 - Monte Carlo Ray Tracing due 6/9.
